# 2play client

### Installation

1. **Clone the repository**

   ```sh
   git clone https://github.com/RMT56-2play/client.git

   ```

2. **Install dependencies**

   ```sh
   # For the Frontend:

   cd client/client
   npm install
   ```

## Access the application

- Frontend: http://localhost:5173

  ```sh
  npm run dev
  ```
